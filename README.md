# Radar Financiero Perú

Aplicación web full stack orientada a integrar, consultar y visualizar información macroeconómica, financiera y regulatoria de fuentes públicas peruanas como BCRP, SBS y SMV.

## Estado del proyecto

Fase inicial de diseño y configuración del entorno.

## Objetivo

Construir una plataforma con backend, frontend, base de datos, procesos de datos, consultas SQL, autenticación, seguridad y documentación técnica.

## Stack previsto

- Frontend: React
- Backend: FastAPI
- Base de datos: PostgreSQL
- Procesamiento de datos: Python
- Control de versiones: Git y GitHub

## Estructura del repositorio

- backend/: API y lógica del servidor.
- frontend/: interfaz web.
- data/: datos originales, intermedios y procesados.
- database/: modelo, migraciones, seeds y consultas SQL.
- docs/: documentación técnica.
- scripts/: scripts auxiliares de ingesta y limpieza.
- tests/: pruebas del proyecto.
